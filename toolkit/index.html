Offline Toolkit Index

This lightweight index is cached by the service worker so your formation tools open instantly, even without a network connection.

  • Shell assets (landing UI, manifest, icons) load Cache-First from the local cache.
  • Toolkit content is refreshed with a Stale-While-Revalidate strategy so updates download quietly in the background.
  • First visit priming ensures subsequent sessions stay fully functional without signing in or reloading remote assets.
